ESERCIZIO BTS_1.1

Rispondere alle domande


1. A cosa serve bootstrap?

   - Bootstrap è un framework front-end gratuito per uno sviluppo web più rapido e semplice, include modelli di design basati su HTML e CSS per tipografia, moduli, pulsanti, tabelle, navigazione, modali, caroselli di immagini e molti altri, oltre a plug-in JavaScript opzionali. Bootstrap ti dà anche la possibilità di creare facilmente design reattivi

2. Cosa devi importare per creare una pagina web con bootstrap?

   - <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

3. Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

   - Il CDN è un gruppo di server distribuiti in più aree geografiche che velocizza la distribuzione dei contenuti web avvicinandoli di più alle posizioni geografiche degli utenti.

4. A cosa serve questo attributo del tag meta? content="width=device-width, initial-scale=1

   - content="width=device-width, initial-scale=1 SERVE per Indicare al browser come controllare le dimensioni della pagina.

5. Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

   - Il Container ha una larghezza fissa per ogni dimensione dello schermo, mentre il Container Fluid si espande per riempire la larghezza disponibile.
